# 🏦 CaPaNiWa ERP - Sistema de Gestión Financiera

CaPaNiWa ERP es una solución integral de contabilidad y finanzas desarrollada con el **Stack MERN** (MongoDB, Express, React, Node.js). Diseñada para ofrecer una interfaz limpia, rápida y funcional para la gestión de activos, pasivos y control de EBITDA en tiempo real.

## ✨ Características Principales

* **📊 Dashboard BI:** Visualización inmediata de Ingresos, Gastos y EBITDA.
* **🧾 Gestión de Facturación:** Registro de asientos contables con generación automática de Hash Verifactu.
* **⚖️ Balance de Situación:** Estructura contable de Activo y Pasivo automatizada.
* **📖 Libro Mayor:** Historial detallado de todas las operaciones conciliadas y pendientes.
* **📥 Exportación Inteligente:** Descarga de reportes contables en formato Excel (.XLSX) con un solo clic.
* **🤖 Copiloto IA (Beta):** Integración con Google Gemini para análisis predictivo de datos contables.

## 🚀 Tecnologías Utilizadas

* **Frontend:** React.js, Axios, XLSX (SheetJS).
* **Backend:** Node.js, Express.
* **Base de Datos:** MongoDB & Mongoose.
* **IA:** Google Generative AI (Gemini SDK).
* **Estilos:** CSS3 nativo con arquitectura Sidebar-Main.
